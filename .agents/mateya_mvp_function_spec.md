## A. 홈

### 인기 급상승

| 항목 | 내용 |
|---|---|
| 화면 | 홈 |
| 기능 | 추천 모임 노출 |
| 설명 | 조회수가 가장 높은 인기 콘텐츠를 표시한다. |
| 노출 정책 | 상단 1개 노출 |
| 대상 | 유저 |

### 함께할 수 있는 경험

| 항목 | 내용 |
|---|---|
| 화면 | 홈 |
| 기능 | 모임 목록 조회 |
| 설명 | 참여 가능한 경험 목록을 카드 리스트로 제공한다. |
| 정렬/노출 정책 | 인기 급상승 상단 1개를 제외한 나머지를 조회수순으로 노출하며, 그 아래 5개를 표시한다. |

#### 카드 표시 정보

- 카테고리
- 모임 이름
- 시작 시간 ~ 종료 시간
- 참여 인원 / 전체 인원
- 비용
- 대표 사진
- 리뷰 평점
  - 클래스인 경우만 표시

### 모임 상세 이동

| 항목 | 내용 |
|---|---|
| 기능 | 모임 상세 이동 |
| 설명 | 모임 카드를 선택하면 상세 페이지로 이동한다. |

## B. 둘러보기

### 검색

| 항목 | 내용 |
|---|---|
| 화면 | 둘러보기 |
| 기능 | 모임 검색 |
| 검색 기준 | 이름, 장소 |
| 페이지네이션 | 무한 스크롤링 페이지네이션 |
| 기본 조회 사이즈 | 20개 |

### 카테고리 선택

| 항목 | 내용 |
|---|---|
| 기능 | 카테고리 선택 |
| 선택 방식 | 단건 선택 가능 |

#### 카테고리 목록

1. 전통문화
2. 스포츠/액티비티
3. 지역축제
4. 음식체험
5. 언어교환
6. 관광/산책
7. 공예
8. 기타

### 필터

검색에 대한 필터를 제공한다.

#### 1. 정렬

| 항목 | 내용 |
|---|---|
| 필수 여부 | 필수 선택 |
| 선택 방식 | 택 1 가능 |
| 기본값 | 추천순 |
| 옵션 | 추천순, 인기순, 최신순, 마감임박순, 가까운 거리순 |

#### 2. 카테고리

| 항목 | 내용 |
|---|---|
| 필수 여부 | 최소 1개 필수 선택 |
| 기본값 | 전체 |
| 선택 방식 | 전체 선택 시 전체 적용, 그 외 단건 또는 복수 선택 가능 |
| 옵션 | 전체, 전통문화, 스포츠·액티비티, 지역축제, 음식체험, 언어교환, 관광·산책, 공예, 기타 |

#### 3. 참가 대상

| 항목 | 내용 |
|---|---|
| 필수 여부 | 필수 선택 |
| 기본값 | 누구나 |
| 선택 방식 | 여러 개 선택 가능 |

#### 4. 언어

| 항목 | 내용 |
|---|---|
| 필수 여부 | 필수 선택 |
| 기본값 | 사용자 프로필의 모국어 |
| 선택 방식 | 여러 개 선택 가능 |
| 추가 로딩 | `+ 24개 언어 더보기` 버튼을 눌러 언어 체크박스를 추가 로딩할 수 있다. |

#### 5. 지역

| 항목 | 내용 |
