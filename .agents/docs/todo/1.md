
회원가입 필수 약관 상세보기 화면 구현

회원가입 화면에 필수 약관 동의 항목이 있습니다.

다음 요구사항에 맞게 구현해주세요.

약관 항목

모든 항목은 필수 동의 항목입니다.

1. [필수] 서비스 이용약관
2. [필수] 개인정보 제3자 제공 동의
3. [필수] 위치기반 서비스 이용약관
4. [필수] 만 14세 이상 확인

기능 요구사항

1. 각 약관 항목을 터치하면 해당 약관 상세 화면으로 이동해야 합니다.
2. 상세 화면에서는 해당 약관의 전체 내용을 확인할 수 있어야 합니다.
3. 뒤로가기 또는 닫기 버튼을 누르면 회원가입 화면으로 복귀해야 합니다.
4. 복귀 시 기존 입력값과 체크 상태는 유지되어야 합니다.
5. 상세 화면은 약관 조회 전용이며, 동의 체크는 회원가입 화면에서만 처리합니다.
6. 모든 필수 약관에 동의해야 회원가입 진행 버튼이 활성화됩니다.

구현 요구사항

* 약관 내용은 실무적으로 자연스럽게 작성해주세요.
* 약관 내용은 하드코딩하지 말고 Model 또는 JSON 형태로 관리해주세요.
* 약관이 추가되더라도 화면 코드를 크게 수정하지 않도록 데이터 기반으로 렌더링해주세요.
* 뒤로가기 시 회원가입 화면 상태가 초기화되지 않도록 처리해주세요.
* 모바일 UX를 고려하여 긴 약관 내용은 스크롤 가능하게 구현해주세요.
* 접근성을 고려하여 체크박스, 약관 제목, 상세보기 버튼의 의미가 명확하게 전달되도록 구현해주세요.
* 코드 작성 전 생성할 파일 구조와 구현 계획을 먼저 제시해주세요.

상세 약관 내용에 포함할 내용

서비스 이용약관

* 서비스 목적
* 회원가입 및 계정 관리
* 서비스 이용 조건
* 금지 행위
* 서비스 중단 및 변경
* 책임 제한
* 문의처

개인정보 제3자 제공 동의

* 제공받는 자
* 제공 목적
* 제공 항목
* 보유 및 이용 기간
* 동의 거부 권리 및 불이익

위치기반 서비스 이용약관

* 위치정보 수집 목적
* 위치정보 이용 범위
* 위치정보 보관 기간
* 위치정보 제3자 제공 여부
* 위치정보 이용 동의 철회 방법
* 위치정보 보호 책임자 또는 문의처

만 14세 이상 확인

* 사용자가 만 14세 이상임을 확인하는 내용
* 만 14세 미만은 회원가입이 제한된다는 안내
* 허위 확인 시 서비스 이용 제한이 있을 수 있다는 내용
